Soma.ai is a B2B SaaS for automated marketing targeting small local businesses (pharmacies, pet shops, fashion, cosmetics, etc.).

**Core features:** AI-generated cards/posts, automatic scheduling and publishing to Instagram/Facebook, WhatsApp messaging via Evolution API, video generation via Gemini (BYOK).

**Two panels:** Company Panel (business owner) and Admin Panel (Isaias, the operator).

**Plans:** Starter (R$39.90/mo) and Pro (R$69.90/mo) with feature gating.

**Why:** Small businesses don't have time/knowledge for social media. Soma.ai automates everything with minimal user interaction.

**How to apply:** All features, UI, and flows should be designed for simplicity and automation. The target user is non-technical.
